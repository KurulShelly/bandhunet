# BandhuNet - Sistem Pelacakan Alumni

Aplikasi web untuk melakukan pelacakan data alumni berdasarkan nama, program studi, dan status pekerjaan.

## Teknologi
- Laravel
- MySQL
- Bootstrap

## Link Aplikasi
https://bandhunet-production.up.railway.app
